Para este ejercicio se declararon las variables necesarias para abrir la tubería, gestionar los archivos, los bytes leídos, los descriptores y el buffer. Se decidió utilizar variables intermedias para almacenar los resultados de las operaciones de abrir, leer y escribir, en lugar de usarlas directamente en los condicionales. Esto mejora la legibilidad del código y permite una mejor comprensión de cada operación. Los bucles while se implementaron de forma que iteran continuamente (while true) hasta que se hayan leído y transferido todos los bytes necesarios, utilizando una condición break para finalizar el bucle de manera controlada una vez completada la transferencia.
